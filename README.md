# About Me:
I'm a frontend developer currently building a client-facing website with React, TailwindCSS, HTML, and CSS — translating design intent into clean, responsive interfaces that actually work.<br>
I have a genuine eye for design aesthetics, which means I don't just write code that functions — I write code that looks right. I'm actively looking to collaborate on frontend projects where that sensibility matters.<br>
Beyond the browser, I'm working through one of the more humbling exercises in modern ML: implementing GPT-2 — a 124-million parameter transformer — entirely from scratch. No shortcuts, no wrappers. It's a learning project, and the point is to understand what's happening at every layer, not just to ship something.<br>
I'm also deliberately investing in prompt engineering, because the ability to communicate precisely with AI systems is quickly becoming as valuable as any other technical skill. I'd rather learn it properly than assume I already know it.
